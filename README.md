# p5.js + Next.js Template

This is a template for a p5.js + Next.js project using TypeScript and Tailwind CSS styles.

## Development
### Prerequisites
- node
- pnpm

### Getting Started

Use the template this template to create a new project.

Install all dependencies:
```bash
pnpm i
```

Run the project:
```bash
pnpm dev
```
